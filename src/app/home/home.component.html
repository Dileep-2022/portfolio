  • To-Do List Application

    Project Overview:

    The To-Do List application is a robust tool designed to help users manage their tasks efficiently. Built using Angular for the frontend and ASP.NET for the backend, this application allows users to add, edit, and delete tasks, providing an intuitive interface that enhances productivity and organization.

    Tools and Technologies Used:

    • Angular: The primary framework for building the frontend, enabling modular development and efficient data binding.
    • ASP.NET: Used as the backend framework to handle API requests, manage data, and facilitate interactions with the database.
    • HTML: Employed for structuring the application’s user interface, ensuring clear and accessible content.
    • CSS: Used for styling the application, creating a visually appealing layout that is user-friendly and responsive.

    Key Features:

    • Task Management: Users can easily add new tasks, mark them as completed, or delete them, all through a streamlined interface.
    • API Integration: Utilizes ASP.NET to create RESTful APIs for task management, allowing for seamless communication between the frontend and backend.
    • JWT Implementation: JSON Web Tokens (JWT) are used for secure user authentication, ensuring that users can safely access their tasks.
    • Persistent Storage: Cookies are used to store the token and user ID, maintaining session state across browser sessions.
    • Responsive Design: The application is designed to be responsive, allowing users to access it seamlessly on various devices, including desktops and mobile phones.

    Impact:

    This project significantly enhanced my technical skills in web development, particularly in using Angular and ASP.NET together. I gained valuable experience in creating interactive user interfaces, managing API requests, and implementing secure authentication methods with JWT. Working on this To-Do List application deepened my understanding of full-stack development and improved my ability to build user-centric applications.
